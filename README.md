## Technologies
This project is created with :
- **MVVM Architecture**
- **Room Database**
- **Dagger Hilt (Dependency Injection)**
- **ViewModel**
- **View Binding**
- **Coroutine**
- **Flow**
- **LiveData**
- **Navigation Components**
- **Night Mode**
- **Testing Framework**
  - **Mockito**
  - **Espresso**
